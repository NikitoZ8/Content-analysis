# Анализ данных о репетиторах

## Описание проекта

Скрипт выполняет комплексный анализ данных о репетиторах, содержащихся в файле **«Образ репититора.xlsx»**. В результате работы скрипта формируются:
* отчёт в формате Excel с таблицами статистических данных;
* набор визуализаций (графиков) в папке `charts`.

## Функциональность

Скрипт выполняет следующие задачи:
1. **Чтение данных** из Excel-файла.
2. **Расчёт числовых статистических показателей** (среднее, медиана, стандартное отклонение и др.) для столбцов: «Возраст», «Стаж», «Оценка», «Количество отзывов», «Стоимость часа».
3. **Подсчёт частот** значений в категориях:
   - платформы;
   - пол;
   - социальный статус;
   - формат занятий (офлайн/онлайн).
4. **Групповой анализ** средней и медианной стоимости часа по категориям:
   - пол репетитора;
   - платформа;
   - формат занятий.
5. **Анализ текстовых данных** (столбцы «Достижения», «Особенности работы»):
   - токенизация русского текста с фильтрацией стоп-слов;
   - подсчёт топ-20 наиболее частых слов.
6. **Генерация отчётов и графиков**:
   - сохранение таблиц в Excel (`content_analysis_report.xlsx`);
   - создание трёх графиков в папке `charts`:
     - распределение репетиторов по платформам;
     - медианная стоимость часа по платформам;
     - топ слов в текстовых полях.

## Требования

Для работы скрипта необходимы следующие библиотеки Python:
* `pandas` — работа с таблицами и данными;
* `matplotlib` — построение графиков;
* `openpyxl` — чтение/запись Excel-файлов;
* `pathlib` — работа с путями к файлам (встроен в Python 3.4+);
* `re` — регулярные выражения (встроен в Python);
* `collections` (Counter) — подсчёт частот слов (встроен в Python);
* `shutil` — работа с файловой системой (встроен в Python).

## Инструкция по запуску

1. Убедитесь, что файл **«Образ репититора.xlsx»** находится в той же папке, что и скрипт `content_analysis.py`.
2. Активируйте виртуальное окружение (если используется) [команда: `venv\Scripts\activate` (Windows) или `source venv/bin/activate` (Linux/macOS)].
3. Запустите скрипт командой:
   ```
   python content_analysis.py
   ```
4. Дождитесь завершения работы скрипта (появится сообщение «Готово.»).
5. Найдите результаты в текущей папке:
   - файл `content_analysis_report.xlsx` (отчёт);
   - папка `charts` с графиками.

## Структура выходных данных

### Excel-отчёт (`content_analysis_report.xlsx`)
Содержит следующие листы:
* **Статистика** — числовые показатели по основным столбцам;
* **Платформы** — распределение репетиторов по платформам;
* **Пол** — распределение по полу;
* **Соц статус** — распределение по социальному статусу;
* **Формат** — распределение по формату занятий;
* **Среднее по полу**, **Среднее по платформе**, **Среднее по формату** — средняя стоимость часа по категориям;
* **Медиана по полу**, **Медиана по платформе**, **Медиана по формату** — медианная стоимость часа по категориям;
* **Топ слов** — топ-20 слов из текстовых полей.

### Папка `charts`
Содержит три графика:
* `platform_counts.png` — количество репетиторов по платформам;
* `median_price_by_platform.png` — медианная стоимость часа по платформам;
* `top_words.png` — топ слов в текстовых полях (с ротацией подписей для удобства чтения).

## Возможные ошибки и их решение

Если скрипт не работает:
1. **Ошибка импорта библиотек** — установите недостающие библиотеки командой:
   ```
   pip install pandas matplotlib openpyxl
   ```
2. **Файл не найден** — проверьте, что «Образ репититора.xlsx» находится в той же папке, что и скрипт.
3. **Проблемы с кодировкой** — убедитесь, что файл Excel сохранён в кодировке UTF-8.
